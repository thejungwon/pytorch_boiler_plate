# pytorch_boiler_plate

## Components
- `main.py`(WIP) : 전체 훈련과정 관리
- `model.py` : 모델 코드
- `optimizer.py` : Custom Optimizer를 사용할 시
- `scheduler.py` : scheduler 정의
- `trainer.py` : 훈련과정
- `dataloader.py` : dataset을 어떻게 Load할지
- `metric.py` : 성능 측정 metric
- `utils.py` : 각종 함수들
- `config.yml` : 각종 파라미터들
